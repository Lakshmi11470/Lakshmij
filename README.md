# Acute Lymphoblastic Leukemia Detection

## Description
This project detects Acute Lymphoblastic Leukemia (ALL) using a deep learning model based on ResNet-50 architecture trained on blood cell images.

## Features
- Image classification using CNN (ResNet-50)
- Detects leukemia from blood cell images
- Provides prediction results

## 🛠 Technologies Used
- Python
- TensorFlow / PyTorch
- OpenCV
- NumPy

## 📂 Dataset
Due to large size, the full dataset is not uploaded.  
Sample images for each stage are included in the repository.

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/lakshmi11470/Lakshmij.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python app.py

## 📊 Output
- Predicts whether leukemia is detected or not based on input image
